# Odin Calculator

## Purpose

Created as a project for [The Odin Project](https://www.theodinproject.com/)

## Functionality

This calculator is relatively basic in how it functions. You can do basic operations (addition, subtraction, multiplication, division), as well as the modulo operator (while normally this button is used for percentages, I thought it would be fun to have a modulo operator as most calculators do not). The backspace button removes a character from the equation, the clear button completely clears the equation and answer fields, and you can change the polarity of your numbers (positive and negative) as well as work with decimals.

The task for this calculator was to simply have it do basic equations, therefore if an operator is already within the equation, pressing another one will simply evaluate the equation first, replace the equation with the answer, and apply the new operator to this.

The calculator automatically formats decimals and long values to fit within the calculator's screen. If an equation is getting too long, the equation bar will start to scroll, automatically displaying the new values typed in, but the equation bar is fully scrollable with a mouse wheel when hovered over.
